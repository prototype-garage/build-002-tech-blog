# Theo's Tech Log MVP

세련된 다크 모드 기반의 개인 기술 블로그 랜딩 페이지 MVP 프로젝트입니다.

## 주요 기능
- **다크 모드 UI**: 현대적이고 깔끔한 디자인 적용.
- **게시글 조회/상세**: 메인 목록과 상세 페이지 전환 지원.
- **게시글 작성**: 클라이언트 사이드 상태 관리를 통한 실시간 글 작성 시뮬레이션.
- **구독 기능**: 이메일 구독 폼 및 알림 피드백.

## 기술 스택
- React
- Vite
- CSS (Custom Variables)
- GitHub Actions (Auto Deploy to GitHub Pages)

## 실행 방법
```bash
npm install
npm run dev
```

## 배포
GitHub Pages를 통해 자동으로 배포됩니다.
